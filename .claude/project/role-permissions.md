# Enterprise Role Permission Matrix

---

## Super Admin

- Full system configuration
- Manage workflows
- Manage SLA policies
- Manage automation rules
- Manage integrations

---

## Admin

- Manage tickets
- Manage users
- Manage masters
- View reports

---

## Agent

- Handle assigned tickets
- Update ticket status
- Add comments

---

## End User

- Create ticket
- Track own tickets
- View notifications
