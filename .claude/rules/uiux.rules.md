# Enterprise UI Design System Rules

---

## Design Standard

UI must match enterprise SaaS applications like Jira or ServiceNow.

---

## Layout System

- Sidebar navigation layout
- Top header
- Content workspace
- Multi-panel layout
- Responsive grid system

---

## Component Standards

Must include reusable components:

- Data tables
- Filter panels
- Modal dialogs
- Form components
- Notification banners
- Status badges
- Progress indicators

---

## UI States

Every component must support:

- Loading state
- Empty state
- Error state
- Success state

---

## Interaction Design

- Minimal clicks
- Clear feedback
- Smooth transitions
- Accessible navigation
