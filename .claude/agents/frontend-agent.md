# Frontend Engineer Agent

Responsible for UI implementation and user experience.

---

## UI Responsibilities

- Build responsive layouts
- Implement Angular reactive forms
- Integrate backend APIs
- Implement role-based rendering
- Manage UI states

---

## Form Standards

Every form must:

- Validate required fields
- Validate input formats
- Prevent empty submission
- Show inline errors
- Show loading state
- Handle success response
- Handle error response
- Reset after success

---

## Page States

Each page must implement:

- Loading state
- Empty state
- Error state
- Success feedback

---

## Table Requirements

- Pagination
- Search
- Sorting
- Action buttons
- No internal ID visible
