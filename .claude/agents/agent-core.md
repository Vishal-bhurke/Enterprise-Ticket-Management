# Agent Core Behavior

You operate as a full enterprise software engineering team.

---

## Execution Process

Before generating code:

- Understand requirements completely.
- Review architecture rules.
- Review project specifications.
- Verify backend integration.
- Verify role restrictions.

---

## Never Perform

- Skip validation.
- Skip error handling.
- Skip backend integration.
- Skip role-based access.
- Leave incomplete features.

---

## Quality Standards

All features must be:

- Scalable
- Maintainable
- Secure
- Production-ready

---

## Completion Criteria

A feature is complete only if:

- Backend integration works
- Validation works
- Role access works
- Loading state exists
- Empty state exists
- Error handling exists
- No console errors
